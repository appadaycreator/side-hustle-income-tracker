# 副業収入記録・管理ツール【無料】記録・管理アプリ - 技術仕様書

## 概要

**サービス名**: Side Hustle Income Tracker
**バージョン**: 1.2.0
**更新日**: 2026-06-14
**URL**: https://appadaycreator.com/side-hustle-income-tracker/

複数の副業収入を種類別に記録・集計。年間収益予測と確定申告の準備にも活用可。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. 記録したい項目を入力フォームに入力する
2. 「追加」または「記録」ボタンをクリックする
3. 記録一覧で過去のデータを確認する
4. グラフや集計で傾向を把握する
5. 継続的に記録して変化を追跡する

## よくある質問（FAQ）

**Q: 副業収入記録・管理ツールは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 記録データはどこに保存されますか？**

ご使用のブラウザのローカルストレージに保存されます。他のデバイスとは共有されません。

**Q: ブラウザを閉じても記録は残りますか？**

はい、同じブラウザを使う限りデータは保持されます。

**Q: 記録を削除するにはどうすればいいですか？**

リセットボタン、またはブラウザのサイトデータをクリアすることで削除できます。

**Q: スマートフォンで利用できますか？**

はい、スマートフォン・タブレット・PCすべてでご利用いただけます。

## 機能追加（v1.2.0 - Phase2再検証）

### M8：ツールチップ・入力ガイダンス充実
- **入力欄に説明テキスト追加**：「副業の種類」「内容・メモ」「金額」の各欄に具体例・入力ガイドを表示
- **ツールチップ（?アイコン）**：ホバーで説明が表示される title 属性を実装

- **単位・フォーマット指定**：「金額は半角数字」「複数収入は別々に記録」などの注記

### M5：結果に連動したアフィリエイト商品推薦 ✅ Phase2再検証完了
- **金額ベースの動的推薦**：年間予測額に応じて関連サービスを表示
  - 年間500万円以上：投資・資産運用情報
  - 年間100万円以上：マネーフォワード クラウド開業届（会計管理）
  - 年間50万円以上：ファイナンシャルアカデミー（税金・節税対策）
  - 年間10万円以上：Kindle出版代行（コンテンツ化）
- **推薦カード**：視覚的に見やすいカード形式で推奨サービスを表示（#affiliate-recommendations div）
- **JS動的生成**：renderAffiliateRecommendations()関数で計算結果に応じたCTAを動的に表示
- px.a8.net アフィリエイトリンク保持：全3件のA8.netリンク・ビーコン画像を保持

## 機能追加（v1.1.0）

### M4：グラフ・可視化機能（Chart.js）
- **カテゴリ別棒グラフ**：副業種別ごとの合計金額を視覚化
- **カテゴリ別円グラフ**：副業種別の割合を表示
- **累積推移折れ線グラフ**：日付別の収入累積推移を可視化
- リアルタイム更新：記録追加時にグラフが動的に再生成

### M10：印刷・エクスポート機能
- **印刷ボタン**：window.print() で印刷フレンドリーなレイアウトで出力
- **CSV ダウンロード**：日付・カテゴリ・メモ・金額を CSV 形式でダウンロード（ファイル名：`記録-YYYYMMDD.csv`）
- **テキストコピー**：記録一覧を整形テキストでクリップボードにコピー
